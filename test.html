Test Plan

Parley is a treasure hunt quiz game designed for the purposes of a group assignment for Computing Skills module at UCLan Cyprus. This product is designed to be a web application, that requests data from Code Cyprus API to implement various functionalities such as request questions, receive answers from the user, calculate score and update leaderboard.

The testing of Parley app aims to verify that the final product executes properly without any errors, as well as validate that the product performs in way that matches the expectations of end-users.

Such methods include functionality testing (unit testing), acceptance testing and a usability assessment.
